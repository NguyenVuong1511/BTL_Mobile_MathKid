# 📱 MathKid - Ứng dụng học toán lớp 1

## 1. Giới thiệu

MathKid là ứng dụng Mobile Android hỗ trợ trẻ em lớp 1 học toán thông qua các bài tập và trò chơi tương tác.

---

## 2. Công nghệ sử dụng

* Android Studio
* Java
* XML Layout
* SQLite Database
* Material Design UI

---

## 3. Chức năng chính

* Đăng ký tài khoản
* Đăng nhập
* Học tập
* Luyện tập
* Làm bài thi
* Lưu điểm số người dùng
* Theo dõi tiến độ học tập

---

## 5. Database

Ứng dụng sử dụng SQLite.

---

## 6. Hướng dẫn chạy dự án

### Bước 1: Clone project

```
git clone https://github.com/yourusername/mathkid-mobile-app
```

### Bước 2: Mở bằng Android Studio

File → Open → chọn thư mục project

### Bước 3: Chạy ứng dụng

Run → chọn Emulator hoặc thiết bị Android
